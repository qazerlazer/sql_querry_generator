# SQL Query Generator

Convert plain English questions into SQL queries using AI.  
Built with Streamlit and Google Gemini 2.5 Flash.

---

## **Features**

- Generate SQL queries from natural language input
- Optional: Provide database schema for more accurate queries
- Interactive and beginner-friendly interface
- Works both locally and deployed on Streamlit Cloud
